Propiedades del padre

  • Display Flex

    Esta propiedad noes permite establecer cual será el contenedor de los elementos que serán flexibles. Puede ser bloque o linea.

    display: flex | display: inline-flex

  • Flex Direction

    Nos permite establecer la dirección en la que se acomodaran nuestros elementos en el eje principal (main size) de forma horizontal (row) o vertical (column).

    Es importante mencionar que el eje principal (main size) siempre dependerá del modo de escritura

    flex-direction: row | column | row-reverse | column-reverse

  • Flex Wrap

    Podemos indicar si todos los elementos estarán en una linea con su máximo ancho posible o establecido (nowrap), o si queremos que respete el ancho de los elementos y estos se posicionan en varias lineas (wrap o wrap-reverse)

    flex-wrap: nowrap | wrap | wrap-reverse

  • Justify Content

    Define como se acomodan los elementos en el eje principal (main size).

    justify-content: flex-end | flex-start | center | space-around | space-between

  • Align Items

    Define como se acomodan los elementos en el eje transversal para cuando tenemos una linea de elementos nowrap

    align-items: flex-start | flex-end | center | stretch | baseline

  • Align Content

    Define como se acomodan los elementos en el eje transversal para cuando tenemos varias lineas de elementos (wrap o wrap-reverse)

    align-content: flex-start | flex-end | center | stretch | space-between | space-around
